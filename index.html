게임 소개

Bone Trail의 첫 목표는 거대한 세계관 소개가 아니라, 짧은 층 하나에서 선택이 바로 손실과 보상으로 돌아오는 감각입니다.

01

짧은 층, 빠른 판단

작은 타일 맵에서 이동, 적과의 거리, 회복 타이밍을 매 턴 판단합니다.

02

뼈 조각 자원

전투와 탐색으로 얻는 bone fragment는 성장과 보상 루프의 중심 자원으로 실험 중입니다.

03

Class route

Guard, Blade 같은 class route로 같은 층에서도 다른 위험 관리 방식을 테스트합니다.

04

Warp descent

층을 정리하고 warp를 타는 짧은 목표를 통해 더 깊은 run 구조로 확장합니다.

개발 방향

마케팅 페이지는 게임을 먼저 보여주고, 기술 상세는 별도 페이지로 분리했습니다.

다음에 좋아져야 할 것

현재 핵심 규칙은 테스트로 검증되어 있지만, 플레이하고 싶게 만드는 UI 밀도와 조작 편의성은 아직 개발 중입니다. 다음 단계는 화면을 더 예쁘게 꾸미는 것보다 실제로 만지는 창과 가방, quickslot, viewport behavior를 개선하는 것입니다.

  • R2bviewport와 scene scale을 안정화해 작은 창에서도 맵과 HUD가 깨지지 않게 만들기.
  • R2cinteractive bag window로 아이템 선택, 상세, 취소, modal dispatch를 실제 조작으로 이동.
  • Demo이동, 전투, 아이템, class active, warp를 30-60초 영상으로 보여줄 수 있게 정리.
